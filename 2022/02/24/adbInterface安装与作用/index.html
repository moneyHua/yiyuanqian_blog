ADB Interface安装与作用

2022/02/24 | 安卓 | 阅读 | 434 字 | 1 分钟

ADB Interface 是一个安卓设备连接 PC（windows 系统）时所需要的驱动程序。

使用安卓设备连接 PC（windows 系统）时，PC 会自动进行驱动程序安装。但驱动安装过程中，可能会出现下图所示情况。由于驱动程序未能正常安装，导致安卓设备不能正常与 PC 进行通讯，所以我们需要手动对该驱动进行安装。

我们可以通过以下步骤进行驱动的手动安装：

  • 打开“计算机管理”，如图 1-2 所示，我们可以看到 ADB Interface 并未正常安装运行。

  • 点击该驱动并右键选择“更新驱动程序软件”，如图 1-3 所示选择“浏览计算机以查找驱动程序软件”。

  • 如图 1-4 所示点击选择。

  • 如图 1-5 所示点击“下一步”。

  • 如图 1-6 所示点击“从磁盘安装”。

  • 如图 1-7 所示点击浏览按钮，获取 ADB Interface 安装文件的位置后点击“确定”。ADB Interface 的获取方式有两种：1、网上获取；2、如果本地有安卓 SDK，可以在 SDK 中获取，获取路径以你实际 SDK 文件夹路径为准，此处为 D:\Android\Sdk\extras\google\usb_driver\下的 android_winusb.inf。

  • 如图 1-8 所示选择并点击“下一步”，可能会出现“更新驱动程序警告”弹窗提示，点击“是”继续安装执行，即可安装完成。

  • ADB Interface 安装好后，连接安卓设备的过程中，可能还会出现识别不出设备的情况。可以如图 1-9 所示打开“计算机管理”判断驱动是否已启用，如未启用则右键选择“启用”。
